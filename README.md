## Librerias utilizadas
- React-router-dom

## Levantar el proyecto

1. Clonar el repositorio 

```
git clone https://github.com/WalRol/e-commerce-cda.git
``` 
2. Instalar las dependencias

```
npm i
```
3. Correr el proyecto

```
npm start
```
